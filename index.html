    Quote Of The Day Viewer (qotd-viewer)

    quotd-viewer is a GUI application that can view the quote of the day from quote of the day servers (which send plain text over port 17). You can check out the Wikipedia page on quote of the day servers here. Qotd-viewer is written entirely in Python, and uses Tkinter for GUI stuff.

    How to clone our git repository:

    $ git clone https://github.com/iandun/qotd-viewer.git
$ cd qotd-viewer

    Thanks to Freeze, we are able to build a UNIX application that does not require python to function. See the README file for more information.
